## Image Guidelines

### Technical Specifications
- Format: JPG for photos, PNG for logos with transparency, SVG for scalable graphics
- Optimization: Compress images for web (use tools like TinyPNG)
- Responsive: Provide high-resolution versions for retina displays
- Alt text: All images have descriptive alt text for accessibility

### Brand Consistency
- Use consistent lighting and color grading
- Maintain professional, high-tech aesthetic
- Ensure all images align with AIMCUT's futuristic brand identity

### Performance
- Lazy loading implemented for all images
- Progressive loading for better user experience
- WebP format alternatives can be added for better compression
